# Making myself out of album covers

2020-03-03 music

I really like music. I also really like well-crafted album covers, especially if it compliments the music on it.

For a while I've wanted to make something out of all these album covers I've looked at over the years.

So a couple of months ago I collected image-files from all the albums I've listened to on Spotify over the last few years, and saw what I could do with them.

I ended up with was a set of scripts that can be used to "upscale" any image (to the best of their abilities) using a given set of album-covers based on what album-cover best matches the given pixel. (One pixel is replaced by one album-cover)

I also thought it would be fitting to recreate a picture of myself using albums that I personally have a connection to. (all covers are selected from the albums I've listened at least once during the last 3-4 years) Although I have to say that I'm not completely happy with all of the covers that were used here, I do value the relative accuracy of the colors used a bit higher. That being said, there are some good ones in there though!

A lot of these have been lying around for a bit, since I tried to figure out some way to share/display them online. I didn't want to distribute them as-is, since most modern photo-viewers have trouble opening them. (this one is 43600x43700, and about 500 MB). At first I wanted to try to write some kind of tiling-solution myself, but after a while I found that there were existing solutions that could handle this for me. (so I don't have to reinvent the wheel)

I also tried to offload the "creation" of the images to the browser (since the images are made up of many copies of the same image), but most browsers didn't like having to load/draw that many images at the same time. I ended up using OpenSeadragon and hosting the GBs of tiles elsewhere. This also means that when viewing the image itself you only have to download a fraction of it. Since only a small part of it is visible in high quality at any time, which means that the image-quality can be decreased when zoomed out. (removing details/information that wouldn't be visible at this zoom-level anyway)
